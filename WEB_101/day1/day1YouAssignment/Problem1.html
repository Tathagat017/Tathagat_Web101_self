< b > tag

<b> tag is used to make text bold or draw attention of the user to the specific content of the webpage. It was previously known as boldface tag. It should used as the last resort when no other tag is available for the use. The use of em tag, header tags and strong tag is preferred over this tag.This is not a self closing tag and requires starting and ending tags

<i> tag

<i> This tag is used to convey alternate voice or mood. It's content is displayed in italic.This is not a self closing tag.

<u> tag

<u> tag is used to underline it's inside content. The line colour can be manipulated via CSS.Typically used with spelling errors or underlying unarticulated underlines.

<select> tag

<select> is used for creating a drop down list. It has name attributed which is used to store the data collected from the user.This is used with option tag to provide options to choose from drop down list. Label tag is used with id attribute to label down the data collected.

<button> tag

<button> tag is used to create a clickable button with text inside,which is not possible with input tag. It can have attributes such as type, format etc describing how the button would be.

<hr> tag

This tag defines a thematic break in an HTML page. It creates a horziontal line seperating conents of the page into thematic sections.It's a self closing tag or empty tag

<br> tag

This tag is used for line break .It's a self closing tag or empty tag.
